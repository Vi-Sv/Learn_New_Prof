## ЦЕХ №1: ИНИЦИАЛИЗАЦИЯ И СБОРКА КАРКАСА (DIL И DAFLIST)

Включаем в работу данного сеанса библиотеку Pandas и создаем промышленную двухмерную матрицу на базе структуры DIL (Dict in List).

### Эталонный код инициализации:
```python
# Включаем в работу данного сеанса библиотеку Pandas
import pandas as pd  # импорт pandas подключает в данный сеанс библиотеку, алиас AS помогает присвоить новое короткое имя для скоростного доступа к библиотеке

# Создали переменную, которой передали адрес на новое значение - таблицу
# Таблица, которую мы фиксируем в памяти, содержит 3 строки с данными (4, если считать шапку).
# О том, что в таблице сидит именно 3 живые строки, свидетельствует синтаксис по созданию списка (list), а именно - квадратные скобки, в которые через запятую передали значения, в нашем случае мы передали "словарь" (dict) из 3 позиций.
# Приведенная ниже конструкция является фундаментом по созданию таблицы в Python - словарь в списке / иначе - dict in list, сокращенно - DIL.
# Железное правило: для корректного создания таблицы следует сразу определить и стандартизировать ключи словаря.
vagon_train = [
    {"number": 5408, "cargo": "зерно", "status": "простой"},
    {"number": 4441, "cargo": "щебень", "status": "в пути"},
    {"number": 3372, "cargo": "зерно", "status": "простой"},
]

# Создали переменную df, которой передали конструкцию (операцию), результат выполнения которой относится к внутреннему инструменту Pandas, а именно:
# Конструкция pd.DataFrame() по шагам:
# 1) pd. - pd это сокращенное имя библиотеки pandas, точка выполняет роль шлюза, указывая библиотеку и точку, мы буквально заныриваем внутрь библиотеки.
# Все то, что идет после точки, является внутренним инструментом нашей библиотеки.
# В нашем случае DataFrame - это инструмент по созданию таблицы. Данному инструменту в скобки мы обязаны передать списки (list) - запомним формулировку DAFLIST (даф лист - комбинация, которая напоминает: передай DataFrame список).
# * Важно уточнить, что можно передавать не только списки, но это Будущие навыки.
df = pd.DataFrame(vagon_train)
```

## ЦЕХ №2: ОПЕРАТОРЫ ИЗВЛЕЧЕНИЯ И СВЕРХСКОРОСТНЫЕ ВЕКТОРНЫЕ МАСКИ

Разбираем физику извлечения данных из памяти компьютера и уничтожаем необходимость написания ручных циклов `for`.

### Код декомпозиции шлюзов:
```python
# В Python квадратные скобки — это команда «дай мне элемент по этому ключу» (иначе: оператор извлечения), при этом, в зависимости от среды, при написании данного кода может отобразиться результат.
# НО квадратные скобки - это не функция "ОТОБРАЗИТЬ", при помощи квадратных скобок мы указываем ключ доступа, а переменной до скобок указываем на "хранилище".
# В list (списке) - это числа от 0 до ...
# В dict (словаре) - это созданные ключи.
# Указывая переменную перед скобками, мы говорим системе ОТКУДА взять этот самый ключ доступа.
df["status"]

# В обход циклам мы можем напрямую применить условие и отобрать данные для целого столбца.
# Чтобы выполнить данную операцию, для начала мы указываем ГДЕ именно мы будем отрабатывать операцию, синтаксис "переменная[]" / df[].
# После - в квадратных скобках мы прописываем условие для любого столбца (столбец = ключ доступа из словаря).
# Чтобы система понимала, к какому именно ключу мы применяем условие, мы и тут обязаны указать принадлежность через конструкцию обращения к столбцу: df["number"] (или df["status"]).
# Именно так и рождается комбинация:
df[df["status"] == "простой"]  # переменная, указывающая на DIL [переменная, указывающая на DIL ["Ключ из DIL"] всякое условие....]
```
